# Infinite Wikipedia Word Explorer

Explore Wikipedia articles in a fun and interactive way! Click on words to discover related articles and keep following the chain of knowledge.

## Features

- Start with a random Wikipedia article.
- Click on any word to fetch its related article.
- Continue exploring the chain infinitely.
- Each clicked word shows the new article in the same place.
- Search for specific words or articles using the search box.
- Random word option for inspiration.

## How to Use

1. Open the project in your browser (preferably via Live Server or `http://localhost`).
2. Click **Start Random** to begin exploring.
3. Click on any word in the article to fetch its related article.
4. Use the **Continue Chain** button or press **Space** to follow a random word from the last article.
5. Clear the chain anytime using **Clear** button.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Wikipedia REST API & Search API
